
To Run The Simulation Program


1-download matlab

2-Find matlab bin path and add it bin path:
    //i.e On Mac
    export PATH=/Applications/MATLAB_R2022b.app/bin/:$PATH 

3-change gcc shared libraries options to match the bin location

4-Start terminal at 'harbor_simulation' folder and run 'make'

3-The output plots and file will be in 'output' folder
